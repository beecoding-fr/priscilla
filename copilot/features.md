# Fonctionnalités de la Plateforme

## A. Admin (Priscilla)

### 1. Compte & Accès

#### 1.1 Authentification
- Se connecter en tant qu'admin
- Gérer son profil (nom, email, mot de passe)

### 2. Dashboard

#### 2.1 Statistiques Générales
- Voir le nombre total de JE (Jeunes Entrepreneurs)
- Voir le nombre total d'ES (Entreprises Solidaires)
- Voir le nombre total de transactions validées
- Voir le nombre de points distribués

#### 2.2 Répartition et Analyse
- Voir la répartition des JE par abonnement (basique / classique / premium)
- Voir les JE les plus actifs (top points / nombre de transactions)
- Voir les ES les plus sollicitées

### 3. Gestion des Entreprises Solidaires (ES)

#### 3.1 Créer une Fiche ES
- **Informations de base :**
  - Nom
  - Description
  - Secteur d'activité
  
- **Localisation :**
  - Adresse
  - Département
  - Région
  
- **Contact :**
  - Contact principal (email, téléphone)
  
- **Métadonnées :**
  - Tags / mots-clés (ex : "restauration", "événementiel", "numérique")
  
- **Gestion des comptes :**
  - Lier un ou plusieurs comptes "Représentant ES" à une ES
  
- **Actions administratives :**
  - Désactiver / archiver une ES (et empêcher de nouvelles transactions)

### 4. Gestion des Jeunes Entrepreneurs (JE)

#### 4.1 Vue d'Ensemble
- Voir la liste des JE

#### 4.2 Détails par JE
- **Informations de base :**
  - Nom
  - Email
  - Secteur d'activité
  - Département
  - Région
  
- **Métriques :**
  - Abonnement actuel
  - Nombre de transactions
  - Nombre total de points
  
- **Actions administratives :**
  - Suspendre un compte JE (fraude, impayé, etc.)

### 5. Configuration Métier (Facultatif V1)

#### 5.1 Système de Points
- Définir les tranches de montant → nombre de points associés
  - Exemple : 
    - 0–1 000€ = 1 point
    - 1 001–5 000€ = 2 points
    - > 5 000€ = 3 points
